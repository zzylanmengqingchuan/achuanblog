# 1.那年年少

2005年2月，我出生在一个普通小镇，度过了快乐的童年时光，幼儿园和小学都是在我们小镇里面上的，感觉每天也挺高兴的，基本上小学生活就这样就要过去了；

但是，一切就是来的这么突然；

# 2.转折点

六年级的时候，当时的我还是非常的幼稚，我的父母为了让我接受更好的教育，我来到了县城实验小学读六年级，来到陌生的环境，确实非常的不适应，县城里面的老师也确实很厉害，周围的同学更是没的说，来到新的环境，我变得非常的平庸；我认识了很多的新朋友，但是现在8年过去了，没有一个联系的；

在乡镇的时候，我从1到5年纪，每一个年纪只有两个班级，到了**实验小学**的6年级，一个年级也是5到6个班级，从各个方面而言，都是比我家乡的那个小镇好很多的，我也非常感谢当时父母的远见，把我送到了实验小学；

实验小学仅仅陪伴了我的六年级这一年的时光，没有过多的回忆，我的小学就毕业了，相当于我的小学横跨两个学校，1-5年级是我们小镇的学校，6年级是县城的实验小学；

# 4.最低迷的时光

下面的这个是我从dy上面找的一个截图，大概这个就是我的高中母校的样子，在这里，我读过了高中三年的时光，不能说美好吧，但是确实由于学习方法，心态各方面的原因，我的高中三年是低迷的，自闭的，消极的一段时光，但是我也是熬了过来；

![image-20250630160342460](https://zzy-1326340203.cos.ap-beijing.myqcloud.com//image-20250630160342460.png)

而母校和我的关联，就在下面的图片里，这个是2年前的我，很模糊，本来是想把名字打码的，后来想想算了吧，微信都实名了，也不在乎这个文章里面的名字了（当然对于熟悉我的人就没必要了）；

这个是我们学校的传统：高考结束之后，来年的9月会把录取情况统计一下进行下面的这个样子年的公布（按照排名），基本就是包含你的照片，分数，录取院校和初中学校（不理解为啥在乎这个），而且这个只会留一年，来年的9月就会换新的；

![image-20250630160357052](https://zzy-1326340203.cos.ap-beijing.myqcloud.com//image-20250630160357052.png)

虽然高中的三年非常的低迷，也没有什么现在保持联系的高中同学，但是我感谢这段经历，我也很感谢高中的老师们的教导和鼓励，因为我认为人总要经历这样的“**至暗时刻**”，每个人都有，只是早晚得问题，有一些事情，是我们早晚都要经历的，努力了，也就没什么可后悔的了，这三年的经历也让我后面更加从容的面对大学里面的各种未知的困难和挑战；

# 5.最自主的时光

大学真的是最自主的时光，我非常喜欢，一个人干自己喜欢的事情，全部都靠自律，我们可以学习自己想要学习的，我们可以作自己想要做的，时间是自己的，说真的，河南高考的应试教育扼杀了我们18岁之前的很多兴趣，大学，我们应该去做一些自己想做的，学习一些自己感兴趣的；

说实在的，对于软件工程这个专业，我并不是很喜欢，但是大学也是在学着，我不喜欢的是这个专业就业时候的具体现状，但是对于一些相关的底层技术，我还是很喜欢研究的，其实在这个时代，我认为编程的技能是需要具备的，但是不一定非要选择计算机专业，即使是18岁之前几乎没有接触过电脑的我，经过两年的摸爬滚打，也是get到了很多的新技能，面对各种问题，也是自己能够解决了，在一点上面，我认为学习计算机还是有用处的；

在大一的时候，我还会参加一些校园活动，但是后来我逐渐认识到，这些校园活动对于我帮助也不是很大，刚开始觉得很新奇的，但是多了的话就觉得一般般了，大二，我几乎没参加什么活动，对于大一入学加入的社团，我也离开了，更加专注于自己喜欢的和对自己有用的技术本领；

至今，也是取得了一些小小的成就，这也是不断去是我进步的重要源泉之一：在这个写作的过程中，也认识了很多新朋友，在一个博主的引荐下，我加入了阿里云社区，成为了专家博主（阿里云还给我发了小礼物和证书，让我高兴了一整天）

# 7.最清晰的职业生涯规划影响者

为什么我清晰的知道我自己需要做什么，这就不得不说我的职业生涯规划的影响着了，是一家IT培训机构，名字我就不说了，我感谢当时的选择，事实证明我没有选错，我在这个培训机构里面学到的不仅仅是真实，更是学习的方法，解决问题的方案，它不仅教会了我知识，更让我看到了目前的就业形势，让我知道作为求职者，我需要具备怎样的技能，并且把这些放到大学的每一个学习；

因为此，我知道自己需要学习什么，什么技能是重要的，怎么样才能四年之后找到一份满意的工作，这个机构教会了我，让我学到了很多，也是这里的老师推荐我写博客，没加入之前，我连博客是什么都不知道，我初入大学校园的时候，也是迷茫的，但是很快找到了方向，这个就是信息差，总之，我还是感谢这个里面的老师的；

在当今的这个时代，我愈发清醒的认识到，明辨是非的能力是多么的最重要，好在自己当时做出了正确的选择，这也不能不说是一种幸运，市面上，各种IT培训机构：黑马，尚硅谷，**教育，还有很多私人的，层出不穷，但是他们的目的只有一个，就是让你报班，我没有说这样有什么错，我只是觉得一定要明辨是非，做出正确的抉择，这个很重要；

# 8.最有成就的决定

我认为做事情，一定要及时的看到正反馈，否则很容易自暴自弃，半途而废，那就是因为没有看到希望，如果看到了正反馈，你自然也就有信息继续做下去了；

那我自己来说，刚开始我的正反馈就来自于写博客别人的点赞和收藏，评论，我认为自己的文章得到了别人的认可，我很高兴，这就是正反馈，激励着我不断的做下去；

后来，我成为了一个知识分享者，我在bilibili创建了账号，开始将自己的所学讲出来，b友的每一次点赞，收藏，评论，关注，就是我的正反馈，让我看到了希望，给我力量，让我继续做下去；

我上传的概率论合集，目前是1.4w播放量，上传的大学物理合集，目前是2w播放量，可能我在这个平台上，还是一个非常小的up，但是每一位朋友的三连和评论，都让我认识到我的努力没有白费，我应该继续做下去；

![image-20250630160442431](https://zzy-1326340203.cos.ap-beijing.myqcloud.com//image-20250630160442431.png)

在这个自媒体的时代，我认为我们每一个大学生都应该尝试自媒体，因为他的成本并不高，我们还年轻，还是有很多的试错的机会的，做一件事情，及时的看到正反馈，并且借此鼓励自己继续做下去，我认为这一点非常的重要；

# 11.关于孔乙己的长衫

到这里，我们来说一下文章的标题：孔乙己的长衫，其实这个我是参考的小灰老师文章的标题，但是又觉得非常有道理，我觉得他说的就是读书人（孔乙己）被学历（长衫）束缚住了，我昨天看了一篇文章，觉得说的特别有道理：

**一个人，要么就彻底无脑，没有知识最好，要么就是‮旦一‬有了知识，就必须向上爬，爬到足以和自身认知相匹配的地位，赚到足够多的钱，很多人标榜自己是孔乙己，就是对于自己的现状不满足罢了，就是因为大家之前都是在卷学历，但是却没有拿到自己想要的结果**

针对于这个问题，网络上充斥着各种各样的声音，仿佛很多人觉得自己都是孔乙己，标榜自己是读书人，但是又不忍心去做一些自认为不符合当前身份的工作；

针对于这个问题我认为每个人都有自己的看法，作为普通的学生，我们就是学好专业技能，把握当下，仅此而已，没有什么需要焦虑的，我们要永远相信一句话：**想都是问题，做才是答案**，提升自己的硬实力才是最重要的，有时候，我们的焦虑，就是因为想得太多，做的太少；
