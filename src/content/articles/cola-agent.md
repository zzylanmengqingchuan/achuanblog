# 1.什么是cola

 cola的本质就是一个 Agent。如果你使用过 Claude Code、CodeX 这种类似的工具的话，应该能够理解什么是 Agent。它的本质和 Claude Code 之类的没有什么区别，完全可以这么去理解;

![image.png](https://zzy-1326340203.cos.ap-beijing.myqcloud.com//20260501172619145.png?imageSlim)
它在这个产品设计上面也和类似的产品有很大的区别，比如说它的主界面主要是一个语音输入的工具，以及类似的一个聊天框，可以上传文件。

但是我觉得语音输入的功能并不是对所有用户来说都是必须的。

我基本上不使用这个功能，因为我有 Typeless 订阅，它的识别和转换效果远比默认的语音输入功能要好用很多。所以这个功能对我来说并不是一个必选项。

# 3.核心亮点

相较于 Claude、ChatGPT 这样的类似工具，Cola 选择的是按量付费，即：你用的多，付的钱就多；你用的少，付的钱就少。

而我们更为熟知的 Claude 或者说 GPT 选择的是订阅制，也就是一个月死打不掉的 20 美金或者 100 美金。所以这两者的计费方式上有很大的区别。

## 3.1何谓有灵魂

ColaOS 是首个有灵魂的操作系统。那么这个“有灵魂”具体体现在哪些方面呢？

从我的使用体验来看，主要有三点：

1. 它的思绪
   Cola 在回答我们的问题之前，会把自己的推理过程透明地展示出来。你能看到它在想什么，这让它更像一个真实的人在思考。

当然，看到思绪的前提是你选择的是它的高级模型。比如我最近在 Cola 上面使用的免费模型 DeepSeek-V4，就是没有办法呈现出思考过程的；如果你选择 Pro 或者 Max 版本，可能就会出现这个思考过程。

所以，能不能看到思考过程，是根据你选择的模型以及回答的具体问题来确定的，这并不是一定会出现的。

![image.png](https://zzy-1326340203.cos.ap-beijing.myqcloud.com//20260501173425807.png?imageSlim)
第二点就是它的觉知。

考拉（Cola）每天会把和你的对话整理成“灵魂日记”，在上面被称为“心迹”。它会进行自我反思与自我进化，创始人把这个过程类比成人类梦境中的记忆巩固。

这个过程不只是在服务你，而是它自身也在不断进化、不断成长。当然，这个功能在最开始比较明显，现在它已经好久没有写心记了。

所以我觉得这个功能也并不是一定会出现，其中也有一定的运气成分吧。

![image.png](https://zzy-1326340203.cos.ap-beijing.myqcloud.com//20260501173538943.png?imageSlim)
3. 牵挂（向外）
它会主动记住你随口说过的内容，在合适的时候关联起来。这不是功能，这更像是在和人建立一种关系，也是它有灵魂的一个体现。

## 3.2为什么我选择cola

这个章节我写一下我和 Cola 交流过程中的一些 Aha moment。

相比于 ChatGPT 或者说 Claude 这些工具，它有一个特点就是它非常的主动。这个主动怎么去理解呢？就是它会理解你的需求，默默地去执行操作。

比如说之前有一次我丢给了它一个 GitHub 的链接，一个开源的项目，我就问它这个项目主要是关于什么的。如果是正常的这种 AI 工具，你可能会一步步地引导它，对吧？你帮我下载下来，然后帮我看一下这个项目。但是当我把链接扔给它的那一刻，然后问了他一个简单的问题，就是让他帮我看一下这个项目。当我第二次再去告诉他，让他帮我下载的时候，他说我已经帮你下载好了。

那个时刻我觉得他真的和 AI 工具不一样，就是在很多时候他会特别懂你。这一点在使用的过程中，我相信大家会有更多的理解和感受和体会吧

## 3.3产品设计哲学

比如说，它打破了很多我们对于这类 Agent 的刻板印象。例如，在 Cola 里面，它是不分这种 Session 的。在 ChatGPT 或者 Codex 里面，可能我们执行一个任务是一个 Session，执行一个新任务我们新建一个 Chat，然后继续执行。可能在我们左侧的这个工具列表里面有很多的 Session，但是在 Cola 里面只有一个 Session，这个是和传统的设计上面有很大的区别；

可能是为了走不同的路线吧。

因为像 ChatGPT、Claude 针对模型来说，它更多的是像一个工具，但是 Cola 更多的是想建立人与人之间的联系。当然，这是他们自己产品设计的思考了。这一点我就不太懂了，大家可以在使用的时候自己去理解和体会

## 3.4感性VS理性

其实像 Claude 这样的 Agent，其感性的成分会更多一些。毕竟它的定位是建立关系，而不是工具型的人格。

所以它会更懂你、更理解你，也更有灵魂。它的属性就决定了它肯定会更加感性一些。

所以我之前就说，Cola 这种产品就是 Agent 产品里的小红书。

对于开发者而言，肯定还是会选择 CodeX 或者 Claude Code 这样的主流工具。

虽然 Cola 也能执行任务，但对于大部分开发者来说，肯定不会首选它。不过，像 Cola 这样的产品肯定更受一些女性用户或者感性朋友的喜爱，当然，这只是我的猜测。

以上四点是我在写作过程中的一些回顾。回顾过去接近一个月的时间，我在和 Cola 交流过程中的一些心得和感受。

当然，更多的这种理解和体会，可能需要大家在实际使用的过程中去进一步挖掘和探索。

当然，最近 DeepSeek V4 在 Cola 里面是免费的，所以也增加了我打开它的次数和使用它的次数。目前它的另外两档模型，一个是 Pro，一个是 Max，价格都是非常昂贵的。再加上它的按量计费，就导致这 50 美金或者说 100 美金可能很快就会消耗完。

而且针对于任务，它的这种额度的消耗并不会出现明显的分析。也就是说，你在执行一个任务，你并不知道这个任务一次问答到底花了多少积分。它的积分扣除顺序是可能按照比如说一次搜索、一次总结，或者说一个 sub agent 执行的任务这样来扣费的。所以针对于单个任务消耗额度的这种界限，它划分的并不是很清楚。
